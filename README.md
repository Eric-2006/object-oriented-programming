# Object-Oriented Programming

A specialized repository dedicated to the implementation and study of Object-Oriented paradigms using Java.

## Project Overview

The primary goal of this project is to apply the core principles of Object-Oriented Programming to solve simple problems. The focus lies on creating modular, reusable, and maintainable code by leveraging:
1. **Abstraction:** Defining clear interfaces and abstract classes to hide implementation details.
2. **Encapsulation:** Protecting data integrity through the use of access modifiers and getters/setters.
3. **Inheritance:** Establishing hierarchical relationships between classes to promote code reuse.
4. **Polymorphism:** Implementing dynamic method binding and overloading to enhance system flexibility.

This repository includes various exercises and systems (such as management tools or simulations) that demonstrate the practical application of these concepts.

## Key Features

* **Class Hierarchy Design:** Well-structured inheritance trees that model real-world entities and logic.
* **Interface-Driven Development:** Use of Java Interfaces to decouple definition from implementation.
* **Data Management:** Implementation of Collections (Lists, Sets, Maps) to handle objects efficiently.

## Technologies Used

* **Language:** Java.
* **Standard:** Clean Code principles and standard Java naming conventions.
